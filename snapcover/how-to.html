How to Use SnapAway

Last updated: 30 March 2026

Quick Start

  1. Download and run the installer from the Download button above.
  2. Allow any OS permissions requested (accessibility/window capture) so SnapAway can control windows.
  3. Open SnapAway from the Start menu or the system tray.

Select a Process

Use the process list in SnapAway's main window to find the application you want to hide. Type part of the app name in the search/filter field to narrow results. Click a process to select it — the selected entry is the target for hide/show actions.

Hide and Restore (Keyboard or Floating Button)

With a process selected, you can hide or restore it in two simple ways:

  • Keyboard shortcut: Open Options → Shortcuts and set a Hide and a Show shortcut. While a process is selected, press the Hide shortcut to hide the window; press the Show shortcut to restore it. Example combos: Ctrl+Alt+H (hide) and Ctrl+Alt+S (show).
  • Floating Hide button: Click the floating Hide button that appears on-screen to hide the selected window immediately. Use the floating control or the tray menu to restore it.

Shortcuts — How to Set

  1. Open SnapAway's Options or Settings.
  2. Find the Shortcuts section and click the record field for Hide; press the key combination you want.
  3. Repeat for Show and save changes.

If a shortcut doesn't register, ensure no other app uses the same combo and that SnapAway has the required OS permissions.

Set a Password (optional)

Protect access to hidden windows with an optional password:

  1. Open SnapAway's Options → Security (or Password).
  2. Choose Set Password, enter a strong password and confirm it.
  3. Optionally enable Require password to restore so hidden windows require the password to be shown.
  4. Save changes. Keep this password safe — if forgotten, contact support or use Restore Defaults (which resets other settings).

Troubleshooting

  • If shortcuts don't work, restart SnapAway and check OS permissions.
  • If the floating button doesn't appear, verify it's enabled in Settings and that the app is allowed to run in the background (tray).
  • Use Restore Defaults in Options to reset shortcuts and settings if needed.
