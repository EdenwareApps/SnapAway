Overview

SnapAway runs locally on the user's device and, by default, does not collect or transmit personal data to external servers.

Data We Do Not Collect

  • We do not automatically collect names, emails, personal files, or project content.
  • We do not send usage or performance logs without prior consent.

Exceptions and Opt-in

If the user opts into optional features (for example, submitting error reports or using an activation service), those flows will be described separately and explicit consent will be requested.

Security

Local access to your files depends on operating system permissions. Keep your system updated and review app permissions.
